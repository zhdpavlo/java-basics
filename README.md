# Object-Oriented Programming (OOP) in Java

## Allgemeine Beschreibung

- Object-Oriented Programming (OOP) ist ein Programmierparadigma, bei dem Programme mithilfe von Objekten aufgebaut werden.
Ein Objekt ist eine Kombination aus Daten (Attributen) und Funktionen (Methoden).

- Java ist eine stark objektorientierte Programmiersprache und eignet sich besonders gut, um OOP-Konzepte zu lernen.

## Zentrale OOP-Konzepte
### Klasse

- Eine Klasse ist ein Bauplan für Objekte.
Es definiert:

  + Attribute (Variablen)

  + Methoden (Funktionen)

  + Konstruktoren

### Objekt

- Ein Objekt ist eine konkrete Instanz einer Klasse.

#### Attribute

- Attribute beschreiben den Zustand eines Objekts, z. B.:

  - Geschwindigkeit

  - Farbe

  - Gewicht

#### Methoden

-  Methoden beschreiben das Verhalten eines Objekts, z. B.:

  - Daten setzen
  - Daten ausgeben

### Konstruktor

- Ein Konstruktor wird beim Erstellen eines Objekts automatisch aufgerufen.
Er wird benutzt, um Anfangswerte zu setzen.


- Konstruktorüberladung

#### Eine Klasse kann mehrere Konstruktoren haben – mit unterschiedlichen Parametern.
