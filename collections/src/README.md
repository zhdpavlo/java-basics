# Java ArrayList – Iteration und Grundlagen

Dieses kleine Beispielprogramm dient als Übungsaufgabe zum Arbeiten mit der Java-Klasse ArrayList.

## Beschreibung

In diesem Programm wird eine ArrayList<Integer> erstellt und mit Zahlen gefüllt.
Anschließend wird versucht, die Liste mithilfe einer Schleife durchzulaufen und die Werte auszugeben.

## Lernziele

- Kennenlernen mit der Klasse ArrayList

- Hinzufügen von Elementen mit add()

- Erste Erfahrungen mit der Iteration über Collections in Java

## Verwendete Konzepte

- ArrayList
- Schleifen (for)
- Konsolenausgabe
